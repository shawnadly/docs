---

### Change the style

- Open the style menu in the citation tools and choose between over 100\+ citation styles including **APA**, **MLA**, or **Chicago**.
- The entire document updates automatically.

### What changes between styles

| Style   | In-text example             | Reference emphasis                  |
| :------ | :-------------------------- | :---------------------------------- |
| APA     | (Smith, 2023)               | Author (Year). Title. Journal…      |
| MLA     | (Smith 23)                  | Author. _Title_. Journal, Year…     |
| Chicago | (Smith 2023, 45) / footnote | Flexible notes/bibliography options |

**Tips**

- Match the style requested by your course, journal, or team.
- If a field is missing (e.g., page range), be sure to edit the entry by clicking on the reference.
