<AccordionGroup>
  <Accordion title="Cite" defaultOpen icon="quote">
    - **Search **by topic, **filter** by year, **add** custom citations
    - Source and insert references as you write
  </Accordion>
  <Accordion title="Text" icon="letter-text">
    - Promote lines to **H2/H3** for clear structure
    - Use H2 for major sections (e.g., Introduction, Methods)
    - Use H3 for subsections (e.g., Participants, Procedure)
  </Accordion>
  <Accordion title="Lists" icon="list">
    - **Bulleted** lists for ideas, examples, references
    - **Numbered** lists when order matters (steps, procedures)
  </Accordion>
  <Accordion title="Tables" icon="table">
    - Insert small tables for quick data
    - Example columns: **Item · Detail · Source**
    - Keep tables concise for readability
  </Accordion>
  <Accordion title="Images" icon="image">
    - Upload an illustrative image to support a point
    - Add a brief caption under the image (what, why it matters)
  </Accordion>
  <Accordion title="Code" icon="code">
    - Preserve formatting for snippets, pseudocode, or examples
    - Use inline code for short terms or variables
  </Accordion>
  <Accordion title="Inline Styling" icon="bold">
    - **Bold** for emphasis and key terms
    - _Italic_ for titles or subtle emphasis
    - Underline and ~~Strikethrough~~ sparingly for clarity
  </Accordion>
</AccordionGroup>

**Tip:** Combine headings \+ lists \+ short paragraphs to keep sections scannable.
