## Editing

| Action             | Mac                   | Windows / Linux         |
| :----------------- | :-------------------- | :---------------------- |
| Bold               | Cmd **B**             | Ctrl **B**              |
| Italic             | Cmd **I**             | Ctrl **I**              |
| Underline          | Cmd **U**             | Ctrl **U**              |
| Strikethrough      | Cmd **Shift S**       | Ctrl **Shift S**        |
| Undo / Redo        | Cmd **Z** / Cmd **Y** | Ctrl **Z** / Ctrl **Y** |
| Select all         | Cmd **A**             | Ctrl **A**              |
| Copy / Paste / Cut | Cmd **C / V / X**     | Ctrl **C / V / X**      |

## Structure

| Action            | Mac                   | Windows / Linux        |
| :---------------- | :-------------------- | :--------------------- |
| Normal text       | Cmd **Alt 0**         | Ctrl **Alt 0**         |
| Heading 1 / 2 / 3 | Cmd **Alt 1 / 2 / 3** | Ctrl **Alt 1 / 2 / 3** |
| Ordered list      | Cmd **Alt 7**         | Ctrl **Alt 7**         |
| Blockquote        | Cmd **Shift B**       | Ctrl **Shift B**       |
| Inline code       | Cmd **E**             | Ctrl **E**             |

## Autocomplete

| Action                     | Mac / Windows / Linux |
| :------------------------- | :-------------------- |
| Trigger                    | **Cmd/Ctrl /**        |
| Accept suggestion          | **Enter** or **Tab**  |
| Reject suggestion          | **Esc**               |
| Previous / Next suggestion | **← / →**             |
