### Symptom → Fix

| Symptom                                   | Fix                                                                                              |
| :---------------------------------------- | :----------------------------------------------------------------------------------------------- |
| No results for a query                    | Use broader keywords; remove the **year** filter; try author/title terms.                        |
| Wrong style formatting                    | Re-select **APA/MLA/Chicago**; then re-open **References** to verify.                            |
| Duplicate entries in References           | Remove the extra in-text citation pointing at the duplicate, or merge details in **References**. |
| Missing fields (e.g., pages, access date) | Edit the entry in **References** and fill required fields.                                       |
| Custom citation won’t save                | Complete all required fields for that type; check year format.                                   |

**Reminder:** Inserting in-text citations auto-adds to **References**.
