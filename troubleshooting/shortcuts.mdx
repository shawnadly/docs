### Quick checks

- Click inside the editor to ensure it has focus.
- OS layouts can differ; see **Shortcuts** in the footer to confirm bindings.

### Symptom → Fix

| Symptom                   | Fix                                                                      |
| :------------------------ | :----------------------------------------------------------------------- |
| Bold/Italic not applying  | Confirm you’re not inside a code block; try toolbar once to reset state. |
| Heading shortcuts ignored | Use **Cmd/Ctrl Alt 1/2/3**; ensure the line isn’t a list item.           |
| Autocomplete keys ignored | Use **Cmd/Ctrl /** to trigger; **Esc** to clear any open popovers.       |
